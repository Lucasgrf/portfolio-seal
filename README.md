# Portfolio Monorepo

Estrutura organizada em workspaces npm.

## Estrutura
- `packages/web`: app React + Vite + Tailwind (framer-motion, lucide-react).

## Scripts
- `npm run dev` — inicia o app em `packages/web`.
- `npm run build` — build de produção do app.
- `npm run preview` — preview do build.

Use `npm install` na raiz para instalar dependências de todos os pacotes.
